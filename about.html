About

Seorang Ui designer pendatang baru dengan karya kratif dan memiliki banyak keahlian yang dapat mendukung sebagai Ui designer. Saya ingin belajar lebih lanjut tentang Ui Designer dan membagikan ilmu Ui ke lebih banyak orang.

View My Works

Hallo, Saya munchamiina biasah dipanggil mintux. saya adalah junior ui designer, pernah bekerja di PT Cipta Karya Abadi selama 1 tahun. selama bekerja di perusahaan tersebut saya lebih sering memegang project berbasis mobile.

Setelah bekerja di PT Cipta Karya Abadi saya pindah perusahaan ke perusahaan yang bergerak dibidang perbankan. saya diperusahaan perbankan ini selama 2,5 tahun sebelum akhirnya memutuskan untuk keluar. diperbankan saya dipercayai untuk mengelola website mereka.

Mulai dari membuat kontent sampai redesign website tersebut. pada tahun 2021 saya memutuskan untuk freelance dan membuka usaha tersendiri. Selama saya freelance ini banyak pekerjaan dibidang mobile dikarenakan kondisi saat ini yang tidak memperbolehkan keluar rumah.

Goals

Tujuan saya adalah membuat solusi untuk semua kalangan dengan bantuan teknologi yang nyaman, efisien dan juga aman. Maka dari itu saya selalu menjaga beberapa hal penting seperti berikut :

Goals

1 - High Quality

Saya selalu menjaga kualitas agar customer saya puas dan gembira. saya siap membuat ulang dari 0 apabila kualitas hasil tidak sesuai.

2 - Pamper Customers

Pembeli adalah raja, itu salah satu prinsip saya. saya akan melayani customer dengan maksimal sehingga mereka puas dengan pelayanan yang saya berikan.

3 - Remote Monitoring

Setelah layanan sudah jadi saya tidak langsung angkat tangan. saya akan memantau jarak jauh untuk melihat apakah layanan tersebut berjalan dengan lancar dan sesuai.
